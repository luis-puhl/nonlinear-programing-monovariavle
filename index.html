Implemneted Methods

Linear Search **TODO**

Given f, [a, b] and \Delta as step;
Start with x = a;
Each step k is given by x_{k} = x_{k-1} + \Delta;
Evaluate f(x_{k}), if it's value is decreasing, aka: f(x_{k}) < f(x_{k-1}), continue. If not, accept x as optimum.
If desired, the result can be further refined repeting the procedure with \frac{\Delta}{10} as step.

Dichotomic Search **TODO**

Given f, [a, b], \epsilon as evaluation range and l as maximum error;
Until b - a < l do the following:

  • Evaluate x = \frac{b + a}{2}
  • Evaluate X = f(x - \epsilon) and Y = f(x + \epsilon)
  • If X > Y make a = x - \epsilon otherwise, make b = x + \epsilon
When the stoping condition has been reach, the optimum will be x^{*} = \frac{b + a}{2}

Golden Section **TODO**

Given f, [a, b] and l as maximum error;
Also using the constants \alpha = \frac{\sqrt{5} - 1}{2} and \beta = 1 - \alpha
Until b - a < l do the following:

  • Evaluate \mu = a + \beta * (b - a) and \lambda = a + \alpha * (b - a)
  • Evaluate f(\mu) > f(\lambda), if true
    • make a = \mu
    • make \mu = \lambda and save the next evaluatin of f(\mu) doing f_{k}(\mu) = f_{k-1}(\lambda)
    Otherwise
    • make b = \lambda
    • make \lambda = \mu and save the next evaluatin of f(\lambda) doing f_{k}(\lambda) = f_{k-1}(\mu)
When the stoping condition has been reach, the optimum will be x^{*} = \frac{b + a}{2}

Fibonacci Search **TODO**

Given f, [a, b] and l as maximum error;
Also using fibonacci sequence F;
Until b - a < l do the following:

  • Evaluate \alpha = \frac{ F _{n-k-2} }{ F _{n-k} } and \beta = \frac{ F _{n-k-1} }{ F _{n-k} }
  • Evaluate \mu = a + \beta * (b - a) and \lambda = a + \alpha * (b - a)
  • Evaluate f(\mu) > f(\lambda), if true
    • make a = \mu
    • make \mu = \lambda and save the next evaluatin of f(\mu) doing f_{k}(\mu) = f_{k-1}(\lambda)
    Otherwise
    • make b = \lambda
    • make \lambda = \mu and save the next evaluatin of f(\lambda) doing f_{k}(\lambda) = f_{k-1}(\mu)
When the stoping condition has been reach, the optimum will be x^{*} = \frac{b + a}{2}

Bisection **TODO**

Given f, [a, b] and l as maximum error;
Also assuming f'(x) \exists \; \forall x \; \in [a..b];
Evaluate n by solving \frac{1}{2} ^{n} <e; \frac{ l }{ b_{1} - a_{1} };
Starting with k = 1 repeat the following until k = n:

  • Evaluate x_{k} = \frac{a + b}{2}
  • Evaluate f'(x_{k}) and check the following
    • case f'(x_{k}) < 0 make a_{k+1} = x_{k} and b_{k+1} = b_{k};
    • case f'(x_{k}) > 0 make a_{k+1} = a_{k} and b_{k+1} = x_{k};
    • case f'(x_{k}) = 0 stop the process because x^{*} = x_{k}.
When the stoping condition has been reach, the optimum will be x^{*} = x_{k}

Newton **TODO**

Given f, [a, b] and \epsilon as maximum error;
Also assuming f'(x) \; and \; f''(x) \exists \; \forall x \; \in [a..b];
Until \lVert f'(x_{k+1})\rVert < \epsilon or do:

  • Evaluate the next nearest minimum point x_{k+1} = x_{k} - \frac{ f'(x_{k}) }{ f''(x_{k}) }
When the stoping condition has been reach, the optimum will be x^{*} = x_{k}
Notes:
  1. If the function is quadratic there is only one iteration;
  2. This procedure is valid only if f(x) \x^{*} = x_{k}
