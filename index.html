Linear Search **TODO**

Given f, [a, b] and \Delta as step;
Start with x = a;
Each step k is given by x_{k} = x_{k-1} + \Delta;
Evaluate f(x_{k}), if it's value is decreasing, aka: f(x_{k}) < f(x_{k-1}), continue. If not, accept x as optimum.
If desired, the result can be further refined repeting the procedure with \frac{\Delta}{10} as step.

Dichotomic search **TODO**

Given f, [a, b], \epsilon as evaluation range and l as maximum error;
Until b - a < l do the following:

  • Evaluate x = \frac{b + a}{2}
  • Evaluate X = f(x - \epsilon) and Y = f(x + \epsilon)
  • If X > Y make a = x - \epsilon otherwise, make b = x + \epsilon
When the stoping condition has been reach, the optimum will be x^{*} = \frac{b + a}{2}

Golden Section **TODO**

Given f, [a, b] and l as maximum error;
Also using the constants \alpha = \frac{\sqrt{5} - 1}{2} and \beta = 1 - \alpha
Until b - a < l do the following:

  • Evaluate \mu = a + \beta * (b - a) and \lambda = a + \alpha * (b - a)
  • Evaluate f(\mu) > f(\lambda), if true
    • make a = \mu
    • make \mu = \lambda and save the next evaluatin of f(\mu) doing f_{k}(\mu) = f_{k-1}(\lambda)
    Otherwise
    • make b = \lambda
    • make \lambda = \mu and save the next evaluatin of f(\lambda) doing f_{k}(\lambda) = f_{k-1}(\mu)
When the stoping condition has been reach, the optimum will be x^{*} = \frac{b + a}{2}
